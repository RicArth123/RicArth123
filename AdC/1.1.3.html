1.1 Modelos de arquitecturas de computo

1.1.3 De multiprocesamiento

 

Cuando se desea incrementar el desempeño más alla de lo que permite la técnica de segmentación del cauce

(limite teórico de una instrucción por ciclo de reloj), se requiere utilizar más de un procesador para la ejecución del programa de aplicación.

LasCPUde multiprocesamiento:

 

SISO–(Single Instruction, Single Operand ) computadoras independientes

SIMO–(Single Instruction, Multiple Operand ) procesadores vectoriales

MISO–(Multiple Instruction, Single Operand ) No implementado

MIMO–(Multiple Instruction, Multiple Operand ) sistemas SMP, Clusters
