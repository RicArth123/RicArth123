2.2 Estructura de registros

2.2.1 Registros visibles para el usuario

 

¿QUÉ SON LOS REGISTROS VISIBLES PARA EL USUARIO?

Son aquellos que el programador o usuario puede acceder directamente, es decir, pueden ser referenciados en el código de programación. Estos registros son importantes porque proporcionan una forma rápida y eficiente de acceder a los datos y las instrucciones de programación en la CPU.

REGISTROS DE DATOS: Estos registros se utilizan para almacenar datos. En la mayoría de las arquitecturas de computadoras modernas, los registros de datos son de tamaño fijo, lo que significa que pueden almacenar una cantidad fija de bits (por ejemplo, 32 bits o 64 bits).

REGISTROS DE DIRECCIÓN: Se utilizan para almacenar direcciones de memoria. La memoria se puede considerar como una gran cantidad de cajas o casilleros, y cada una de estas cajas o casilleros tiene una dirección única.

REGISTROS DE ESTADO: Se utilizan para indicar el estado actual de la CPU. Por ejemplo, pueden indicar si se ha producido una interrupción, si se ha producido un error o si se ha completado una operación.
